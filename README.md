The Automotive Marketplace is a full-stack web application developed using the MERN Stack (MongoDB, Express.js, React.js, Node.js). This platform connects vehicle sellers and buyers in a simple and efficient way.

Sellers can create listings by uploading their vehicle photos and providing all necessary details such as model, price, and specifications. Buyers can browse available vehicles, view detailed information, and contact sellers directly for further discussions.
